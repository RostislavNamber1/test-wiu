Полное руководство по использованию display-flex
(С примерами)

Статья опубликована 8.10.2021 11:15

display: flex

Для того чтобы увидеть эффект гибких блоков, нам потребуется следующая конструкция.

html 
<div class="d-flex">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
    <div class="item">4</div>
</div>
CSS 
.d-flex{
    display: flex;
    width: 100%;
    min-height: 200px;
}
.item{
    width: 100px;
    height: 100px;
    border: 1px solid #000;
}

Отлично. Наши блоки выстроились в горизонтальную линию.

: flex-start (Значение по умолчанию)

Таким свойством мы можем добиться начального положения блоков. Это значение устанавливается браузером по умолчанию.

.d-flex{
    justify-content: flex-start;
}

justify-content: center

Таким свойством мы можем добиться выравнивания по центру горизонтальной оси.

.d-flex{
    justify-content: center;
}

justify-content: flex-end

Таким свойством мы можем добиться что наши блоки будут в конце родительского блока.

.d-flex{
    justify-content: flex-end;
}

justify-content: space-around

Это свойство нам позволяет выронить наши блоки на равным расстояние друг от друга.

.d-flex{
    justify-content: space-around;
}

justify-content: space-evenly

Свойство space-evenly нам позволяет расположить наши блоки на одинаковом расстояние друг от друга. А по бокам экрана расстояние увеличивается.

.d-flex{
    justify-content: space-evenly;
}

justify-content: space-between

Свойство space-between нам позволяет стирать границы и располагает блоки по всей ширине родительского блока с расстоянием между другими блоками.

.d-flex{
    justify-content: space-between;
}

align-items: center + justify-content: center

Два свойства display flex align-items и justify-content в тандем и очень хорошо справляются в различными задачами. Если в каждое свойство записать center, то блоки будут находиться в центе родительского блока.

.d-flex{
    justify-content: center;
    align-items: center;
}

align-items: center + justify-content: space-around

Зная что делает свойство justify-content: space-around мы добавим align-items: center. И в итоге мы можем получить интересное расположение наших блоков.

.d-flex{
    align-items: center;
    justify-content: space-around;
}

align-items: flex-start (Значение по умолчанию) + justify-content: center

Значением по умолчанию align-items считается flex-start. 

.d-flex{
    align-items: flex-start;
    justify-content: center;
}

align-items: flex-end + justify-content: center

Эти значения вам дадут что наши блоки будут находиться в нижней части блока.

.d-flex{
    justify-content: center;
    align-items: flex-end;
}
