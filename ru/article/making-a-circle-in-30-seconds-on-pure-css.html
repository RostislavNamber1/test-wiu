Делаем круг за 30 секунд на чистом CSS

Статья опубликована 30.09.2021 11:15

Привет!

Ты наверняка не просто тут оказался? И пришёл узнать как создать правильно круг на CSS?

Есть много различных способов это сделать! Но я покажу тебе самый простой и самое главное легкий способ.

1. Давай создадим блок высотой и шириной 200 пикселей. И зададим ему цвет например

html <div class="circle"></div> CSS .circle { width: 200 px;
height: 200 px;
background-color: rgb(16, 118, 207);
}

2. Отлично, но наш блок имеет формы квадрата, а не круга. Для этого в стилях CSS нужно прописать border-radius с внутренним значением 50%

CSS .circle { border-radius: 50%;
}

Отлично мы получили круг на чистом CSS. А для этого нам потребовалось немного кода.

Теперь ты знаешь больше!
