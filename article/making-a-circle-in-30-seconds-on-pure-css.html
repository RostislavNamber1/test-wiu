Making a circle in 30 seconds on pure CSS

The article has been published 30.09.2021 11:15

Hello!

Surely you didn't just happen to be here? And I came to find out how to create a circle correctly in CSS?

There are many ways to do this! But I'll show you the simplest and most importantly the easiest way.

1. Let's create a block with a height and width of 200 pixels. And we'll give it a color.

html <div class="circle"></div> CSS .circle { width: 200 px;
height: 200 px;
background-color: rgb(16, 118, 207);
}

2. Great, but our block is shaped like a square, not a circle. To do this, in CSS styles, you need to register border-radius with an internal value of 50%.

CSS .circle { border-radius: 50%;
}

Great, we got a circle on pure CSS. And for this we needed a bit of code.
