Complete Usage Guide display-flex
(With examples)

The article has been published 8.10.2021 11:15

display: flex

In order to see the effect of flexible blocks, we need the following construction.

html 
<div class="d-flex">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
</div>
CSS 
.d-flex{
  display: flex;
  width: 100%;
  min-height: 200px;
}
.item{
  width: 100px;
  height: 100px;
  border: 1px solid #000;
}

Excellent. Our blocks are lined up in a horizontal line.

justify-content: flex-start (Default value)

With this property, we can achieve the initial position of the blocks. This value is set by the browser by default.

.d-flex{
  justify-content: flex-start;
}

justify-content: center

With this property, we can achieve alignment in the center of the horizontal axis.

.d-flex{
  justify-content: center;
}

justify-content: flex-end

By this property we can achieve that our blocks will be at the end of the parent block.

.d-flex{
  justify-content: flex-end;
}

justify-content: space-around

This property allows us to align our blocks at an equal distance from each other.

.d-flex{
  justify-content: space-around;
}

justify-content: space-evenly

The space-even property allows us to place our blocks at the same distance from each other. And on the sides of the screen, the distance increases.

.d-flex{
  justify-content: space-evenly;
}

justify-content: space-between

The space-between property allows us to erase borders and positions blocks across the entire width of the parent block with the distance between other blocks.

.d-flex{
  justify-content: space-between;
}

align-items: center + justify-content: center

The two display flex properties align-items and justify-content in tandem and do a very good job in various tasks. If you write center to each property, then the blocks will be in the center of the parent block.

.d-flex{
  justify-content: center;
  align-items: center;
}

align-items: center + justify-content: space-around

Knowing what the justify-content: space-around property does, we will add align-items: center. And as a result, we can get an interesting arrangement of our blocks.

.d-flex{
  align-items: center;
  justify-content: space-around;
}

align-items: flex-start (Default value) + justify-content: center

Default value align-items be flex-start. 

.d-flex{
  align-items: flex-start;
  justify-content: center;
}

align-items: flex-end + justify-content: center

These values will give you that our blocks will be at the bottom of the block.

.d-flex{
  justify-content: center;
  align-items: flex-end;
}
